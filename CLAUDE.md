# OpenClaude — Foundation Operations Fork

## What This Is
Open-source Claude Code fork supporting multiple LLM providers. Forked from Gitlawb/openclaude.

## Quick Reference
- **Build:** `bun install && bun run build`
- **Run:** `openclaude` (global command) or `node dist/cli.mjs`
- **Test:** `bun test`
- **Provider:** Local Ollama (qwen2.5:7b) at localhost:11434

## Project Structure
- `src/` — TypeScript source (all changes go here)
- `dist/` — Compiled output (regenerated by `bun run build`)
- `docs/` — Setup guides
- `ARCHITECTURE.md` — Full system architecture

## Environment
Config in `/opt/openclaude/.env` and `/root/.bashrc`:
- `CLAUDE_CODE_USE_OPENAI=1`
- `OPENAI_BASE_URL=http://localhost:11434/v1`
- `OPENAI_MODEL=qwen2.5:7b`
- `OPENAI_API_KEY=ollama`

## Conventions
- Always rebuild after source changes: `bun run build`
- Test with `bun test` before committing
- This is a fork — keep commits clean for potential upstream PRs
- Push to origin (foundationoperations/openclaude), not upstream
